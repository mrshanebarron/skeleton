# Laravel Design Skeleton

Skeleton loading placeholder component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/skeleton
```

## Usage

### Livewire Component

```blade
<livewire:ld-skeleton />
```

### Blade Component

```blade
<x-ld-skeleton />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-skeleton-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-skeleton-views
```
